📄 관상 분석 웹캠 프로그램 GitHub README (마크다운 포맷)🌟 Real-time Face Reading Webcam Program (실시간 관상 분석 웹캠 프로그램)💡 프로젝트 개요이 프로젝트는 파이썬 기반으로 개발된 실시간 관상 분석 시뮬레이션 도구입니다. OpenCV와 MediaPipe를 활용하여 웹캠 스트림에서 사용자의 얼굴을 인식하고, 주요 얼굴 특징점의 비율을 분석하여 재물운, 애정운, 건강운 등 관상학적 해석 결과를 한글로 실시간 출력합니다.⚙️ 주요 기술 스택기술역할Python 3.x핵심 프로그래밍 언어OpenCV웹캠 스트림 제어 및 이미지 처리MediaPipe Face Mesh468개 고정밀 얼굴 특징점 실시간 추출NumPy특징점 좌표 기반 비율 및 거리 계산Pillow (PIL)OpenCV 환경 한글 깨짐 방지 및 폰트 출력 지원✨ 주요 기능실시간 특징점 분석: 웹캠을 통해 얼굴을 인식하고, 눈, 코, 미간 등 주요 부위에 특징점을 시각화합니다.다중 관상 분석:💰 재물운 (코/재백궁): 코 폭 비율을 분석하여 금전 운세를 해석.❤️ 애정운 (눈/전택궁): 눈 주변 영역의 높이를 분석하여 대인 및 애정 운세를 해석.🩺 건강운 (미간/명궁): 미간의 너비를 분석하여 심신 안정도 및 건강 운세를 해석.한글 출력 지원: Pillow 라이브러리를 이용해 분석 결과를 명확한 한글 텍스트로 화면에 출력합니다.
